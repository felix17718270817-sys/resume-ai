# AI Resume Optimizer

An AI resume optimizer built with Streamlit and DeepSeek.

Features:
1. Upload PDF, DOCX, or TXT resumes
2. Analyze job description matching
3. ATS keyword analysis
4. Generate optimized resume suggestions
5. Export Word report
